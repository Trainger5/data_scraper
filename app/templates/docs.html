Learn how to use Email Extractor Pro effectively

Introduction

Email Extractor Pro is a powerful tool designed to scrape emails and phone numbers from various sources on the web. It uses advanced techniques like proxy rotation and user-agent spoofing to bypass detection.

Web Search Extraction

Extract leads by searching keywords on popular search engines.

  1. Go to the Dashboard.
  2. Select the Web Search tab.
  3. Choose a search engine (DuckDuckGo is recommended for best results).
  4. Enter a query (e.g., "Software companies in California" email).
  5. Click Start Search.
Tip: Use advanced search operators like site:linkedin.com or "gmail.com" to refine results.
Google Maps Extraction

Target local businesses using Google Maps.

  1. Select the Google Maps tab on the Dashboard.
  2. Enter the Business Type (e.g., Dentists).
  3. Enter the Location (e.g., London, UK).
  4. Click Extract from Maps.

The tool will find businesses, visit their websites, and extract contact info.

Social Media Extraction

Find profiles and contact info from social networks.

  1. Select the Social Media tab.
  2. Choose a platform (LinkedIn, Twitter, etc.).
  3. Enter keywords (e.g., CEO Tech Startup).
  4. Click Search Social.
Configuration

Customize the behavior of the scraper in the Settings page.

  • Proxies: Enable to hide your IP. Requires proxies.txt file.
  • Threads: Increase for faster speed, decrease for stability.
  • Headless Mode: Run browser invisibly (faster).
